# Stock Portfolio Tracker

## Description
This is a simple Python project that calculates the total investment value of a stock portfolio.

## Features
- User enters stock symbols and quantities.
- Uses predefined stock prices.
- Calculates the total investment value.
- Optionally saves the report to a text file.

## Technologies Used
- Python

## How to Run
1. Run the Python file.
2. Enter stock symbols and quantities.
3. Type "done" to finish.
4. View the portfolio summary.
5. Choose whether to save the report to a file.
